## ğŸ° Tentang Project
`MongoAspNetProject` adalah sistem backend sekaligus prototype **Bakery Management System** berbasis **ASP.NET Core** dan **MongoDB**.  
Aplikasi ini membantu admin toko dalam mengelola data pembeli, produk bakery, stok bahan, dan laporan penjualan secara efisien.  

Sistem ini memiliki desain antarmuka modern dengan warna lembut khas bakery, cocok untuk sistem manajemen toko roti dan kue.
